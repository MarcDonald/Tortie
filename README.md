## To use

- Install using `npm` or `yarn`
- In your app's root add `<TortieGlobalStyles />` e.g.

```jsx
import { TortieStylesGlobal } from 'tortie';

return (
  <>
    <TortieStylesGlobal />
    <App />
  </>
);
```
